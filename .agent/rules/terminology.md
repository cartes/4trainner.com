# Terminology Rules

Para mantener la consistencia en la plataforma 4Trainner, se deben seguir las siguientes reglas de terminología:

## Roles y Actores

- **Personal Trainer**: Es el término estándar para referirse a los instructores, profesores o entrenadores.
    - En la interfaz de usuario (UI), siempre debe mostrarse como "Personal Trainer".
    - Evitar el uso de "Profesor" o "Entrenador" en etiquetas, botones y mensajes dirigidos al usuario.
- **Alumno**: Es el término estándar para los clientes de los Personal Trainers.

## Implementación Técnica

- Aunque la base de datos o el sistema de roles interno pueda usar términos como `profesor` o `trainer` por razones técnicas o de legado, el mapeo hacia la UI debe ser siempre "Personal Trainer".
